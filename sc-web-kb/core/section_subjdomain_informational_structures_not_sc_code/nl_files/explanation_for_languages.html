Под конкретным естественным языком понимается некоторое множество файлов (например, идентификаторов, естественно-языковых пояснений и т.д.), которые используются при диалоге с тем или иным пользователем, режим ведения которого он может выбрать.

В этом смысле некоторые фрагменты, такие как, например обозначения sin, cos, e. и т.п., могут входить в несколько естественных языков, поскольку используются при диалоге, но исторически являться фрагментами другого языка.
