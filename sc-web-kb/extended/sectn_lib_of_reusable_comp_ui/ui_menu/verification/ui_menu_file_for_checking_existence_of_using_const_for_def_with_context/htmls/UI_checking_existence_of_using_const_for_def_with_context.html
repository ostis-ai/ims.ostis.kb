Команда проверки наличия используемых констант у понятий, имеющих определение в рамках контекста предназначена для проверки наличия используемых констант у определений или пояснений для понятий. Первым аргументом команды является знак структуры, из которой будут взяты параметры для проверки. Вторым аргументом является контекст действия, в рамках которого будет осуществляться поиск.
