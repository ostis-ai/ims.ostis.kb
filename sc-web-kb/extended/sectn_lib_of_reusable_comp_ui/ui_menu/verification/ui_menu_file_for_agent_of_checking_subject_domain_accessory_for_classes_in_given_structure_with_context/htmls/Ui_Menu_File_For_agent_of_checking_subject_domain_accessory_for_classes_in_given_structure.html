Команда для проверки указания соответствующей предметной области для классов в рамках заданной структуры с заданным контекстом предназначена для нахождения понятий, в заданной структуре, элементы которой не соотнесены с предметной областью. Первым аргументом команды является структура, элементы которой необходимо проверить. Вторым аргументом является контекст действия, который ограничивает область видимости агента. Пример входного аргумента действия. проверить указания соответствующей предметной области и результат работы агента представлен ниже. В ходе выполнения агента явно указывается связь с scp-программой генерации ответа для агентов верефикации и scp-программой добавления сообщения об ошибке.
