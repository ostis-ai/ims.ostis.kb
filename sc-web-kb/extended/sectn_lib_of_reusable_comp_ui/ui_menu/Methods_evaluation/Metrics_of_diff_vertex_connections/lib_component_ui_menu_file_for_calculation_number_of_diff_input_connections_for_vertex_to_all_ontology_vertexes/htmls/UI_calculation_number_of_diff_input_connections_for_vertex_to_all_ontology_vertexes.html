Команда пользовательского интерфейса для вычисления количества вершин с разными типами входящих связей по отношению ко всем вершинам онтологии предназначена для вычисления количества вершин с разными типами входящих связей по отношению ко всем вершинамонтологии. Единственным аргументом команды является знак онтологии, для которой будет вычисляться значение количества вершин с разными типами входящих связей по отношению ко всем вершинам. Результатом выполнения команды является значение количества вершин с разными типами входящих связей по отношению ко всем вершинам онтологии.
