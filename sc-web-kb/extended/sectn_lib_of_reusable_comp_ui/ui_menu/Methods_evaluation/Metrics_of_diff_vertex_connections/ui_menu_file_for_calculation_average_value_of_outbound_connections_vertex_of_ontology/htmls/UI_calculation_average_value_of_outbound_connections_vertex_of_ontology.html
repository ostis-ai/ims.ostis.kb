Команда пользовательского интерфейса для вычисления среднего числа различных типов исходящих связей вершины онтологии предназначена для вычисления среднего числа различных типов исходящих связей вершины онтологии. Единственным аргументом команды является знак онтологии, для которой будет вычисляться среднее значение. Результатом выполнения команды является значение среднего числа различных типов исходящих связей вершины онтологии.
