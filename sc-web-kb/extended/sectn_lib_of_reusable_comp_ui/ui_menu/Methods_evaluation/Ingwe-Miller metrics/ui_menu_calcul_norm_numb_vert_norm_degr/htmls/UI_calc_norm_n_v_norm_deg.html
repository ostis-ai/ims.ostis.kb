Команда подсчета нормированного количества вершин с нормальной степенью предназначена для подсчета нормированного количества вершин с нормальной степенью онтологии(Степень вершины - сумма всех её входящих и исходящих связей). Нормальная степень вершины онтологии равна 9 (метрики Ингве-Миллера). Единственным аргументом команды является знак онтологии. Результатом выполнения команды является значение нормированного количества вершин с нормальной степенью вершины онтологии.
