Команда подсчета дисперсии степени вершины онтологии предназначена для подсчета дисперсии степени вершины онтологии(Степень вершины - сумма всех её входящих и исходящих связей). Единственным аргументом команды является знак онтологии. Результатом выполнения команды является значение дисперсии степени вершины онтологии.
