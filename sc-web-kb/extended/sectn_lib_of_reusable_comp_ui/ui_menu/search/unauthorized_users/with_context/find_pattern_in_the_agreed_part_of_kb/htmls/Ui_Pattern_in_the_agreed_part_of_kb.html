Команда поиска конструкций для заданного шаблона в согласованной части базы знаний предназначена для нахождения всех конструкций, удовлетворяющих заданному шаблону в рамках согласованной части базы знаний, которая ограничивает область поиска агента. Единственным аргументом команды является шаблон, для которого должны быть найдены все соответствующие ему конструкции. Результатом выполнения команды является множество найденных для этого шаблона конструкций.
