Команда формирования согласованной части базы знаний предназначена для формирования контекста для пользователя, в который включаются:

  • Модель базы знаний , а также все её элементы.
  • Спецификация всех разделов базы знаний.
  • Содержимое всех атомарных разделов, частных по отношению к: предметной части базы знаний, контексту предметной части базы знаний в рамках Глобальной базы знаний, документации компьютерной системы.
