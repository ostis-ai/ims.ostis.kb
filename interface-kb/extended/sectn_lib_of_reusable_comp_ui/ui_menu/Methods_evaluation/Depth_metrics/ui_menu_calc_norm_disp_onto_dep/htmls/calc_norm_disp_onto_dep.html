Команда вычисления дисперсии глубины онтологии по отношению к средней глубине предназначена для вычисления дисперсии глубины онтологии по отношению к средней глубине. Единственным аргументом команды является знак онтологии. Результатом выполнения команды является значение дисперсии глубины онтологии по отношению к средней глубине.
