Команда пользовательского интерфейса для подсчёта количества sc-дуг в структуре c заданным контекстом предназначена для подсчёта количества sc-дуг, принадлежащих структуре в рамках заданного контекста. Первым входным аргументом является структура, для которой определяется количество sc-дуг. Вторым входным аргументом является контекст, в рамках которого осуществляется поиск. Результатом выполнения команды является значение количества sc-дуг, принадлежащих структуре в рамках заданного контекста.
