Команда пользовательского интерфейса для подсчёта количества sc-дуг в структуре предназначена для подсчёта количества sc-дуг, принадлежащих структуре. Единственным входным аргументом является структура, для которой определяется количество sc-дуг. Результатом выполнения команды является значение количества sc-дуг, принадлежащих структуре.
