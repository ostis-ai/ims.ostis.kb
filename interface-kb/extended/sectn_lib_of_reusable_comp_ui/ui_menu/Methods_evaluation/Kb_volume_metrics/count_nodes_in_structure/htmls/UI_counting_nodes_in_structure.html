Команда пользовательского интерфейса для подсчёта количества sc-узлов в структуре предназначена для подсчёта количества sc-узлов, принадлежащих структуре. Единственным входным аргументом является структура, для которой определяется количество sc-узлов. Результатом выполнения команды является значение количества sc-узлов, принадлежащих структуре.
