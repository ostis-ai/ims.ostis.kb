Команда подсчета количества относительных и абсолютных понятий базы знаний предназначена для подсчета количества относительных и абсолютных понятий базы знаний. У данной команды нет аргументов. Результатом выполнения команды является значение количества относительных и абсолютных понятий базы знаний.
