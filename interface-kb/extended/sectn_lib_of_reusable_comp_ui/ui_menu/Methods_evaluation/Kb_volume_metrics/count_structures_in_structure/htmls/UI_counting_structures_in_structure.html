Команда пользовательского интерфейса для подсчёта количества знаков структур в структуре предназначена для подсчёта количества знаков структур, принадлежащих структуре. Единственным входным аргументом является структура, для которой определяется количество знаков структур. Результатом выполнения команды является значение количества знаков структур, принадлежащих структуре.
