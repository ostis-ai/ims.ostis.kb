Команда пользовательского интерфейса для подсчёта количества sc-переменных в структуре предназначена для подсчёта количества sc-переменных, принадлежащих структуре. Единственным входным аргументом является структура, для которой определяется количество sc-переменных. Результатом выполнения команды является значение количества sc-переменных, принадлежащих структуре.
