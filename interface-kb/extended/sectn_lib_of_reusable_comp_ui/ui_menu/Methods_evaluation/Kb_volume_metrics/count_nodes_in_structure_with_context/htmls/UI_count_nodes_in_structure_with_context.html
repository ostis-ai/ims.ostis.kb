Команда пользовательского интерфейса для подсчёта количества sc-узлов в структуре c заданным контекстом предназначена для подсчёта количества sc-узлов, принадлежащих структуре в рамках заданного контекста. Первым входным аргументом является структура, для которой определяется количество sc-узлов. Вторым входным аргументом является контекст, в рамках которого осуществляется поиск. Результатом выполнения команды является значение количества sc-узлов, принадлежащих структуре в рамках заданного контекста.
