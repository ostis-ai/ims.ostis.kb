Команда пользовательского интерфейса для подсчёта количества знаков структур в структуре c заданным контекстом предназначена для подсчёта количества знаков структур, принадлежащих структуре в рамках заданного контекста. Первым входным аргументом является структура, для которой определяется количество знаков структур. Вторым входным аргументом является контекст, в рамках которого осуществляется поиск. Результатом выполнения команды является значение количества знаков структур, принадлежащих структуре в рамках заданного контекста.
