Команда пользовательского интерфейса подсчета среднего числа различных типов входящих связей вершины онтологии предназначена для вычисления среднего числа различных типов входящих связей вершины онтологии. Единственным аргументом команды является знак онтологии, для которой будет вычисляться среднее значение. Результатом выполнения команды является значение среднего числа различных типов входящих связей вершины онтологии.
