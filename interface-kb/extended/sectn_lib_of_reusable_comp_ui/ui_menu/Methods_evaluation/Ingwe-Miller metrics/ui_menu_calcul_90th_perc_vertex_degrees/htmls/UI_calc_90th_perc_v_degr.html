Команда подсчета 90-ой процентили степени вершины онтологии предназначена для подсчета 90-ой процентили степени вершины онтологии(Степень вершины - сумма всех её входящих и исходящих связей). Единственным аргументом команды является знак онтологии. Результатом выполнения команды является значение 90-ой процентили степени вершины онтологии.
