Команда подсчета средней степени вершины онтологии предназначена для подсчета средней степени вершины онтологии(Степень вершины - сумма всех её входящих и исходящих связей). Единственным аргументом команды является знак онтологии. Результатом выполнения команды является значение средней степени вершины онтологии.
