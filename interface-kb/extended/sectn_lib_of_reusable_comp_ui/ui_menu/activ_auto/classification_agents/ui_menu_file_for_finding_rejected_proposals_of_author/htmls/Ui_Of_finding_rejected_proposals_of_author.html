Команда пользовательского интерфейса для поиска всех отклоненных предложений разработчика предназначена для поиска отклоненных предложений. Единственным аргументом команды является разработчик, для которого необходимо найти все отклоненные предложения. Результатом выполнения команды является вывод на экран всех отклоненных предложений разработчика.
