Команда пользовательского интерфейса для поиска всех черновиков разработчика предназначена для поиска всех черновиковразработчика. Единственным аргументом команды является разработчик, для которого необходимо найти все черновики. Результатом выполнения команды является вывод на экран всех черновиков разработчика.
