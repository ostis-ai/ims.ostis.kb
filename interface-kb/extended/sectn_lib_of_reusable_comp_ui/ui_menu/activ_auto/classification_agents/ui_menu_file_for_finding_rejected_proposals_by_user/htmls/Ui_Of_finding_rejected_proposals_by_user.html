Команда пользовательского интерфейса для поиска отклоненных предложений конкретным пользователем предназначена для поиска отклоненных предложений. Единственным аргументом команды является пользователь, для которого необходимо найти все отклоненные предложения. Результатом выполнения команды является вывод на экран всех отклоненных конкретным пользователем предложений.
