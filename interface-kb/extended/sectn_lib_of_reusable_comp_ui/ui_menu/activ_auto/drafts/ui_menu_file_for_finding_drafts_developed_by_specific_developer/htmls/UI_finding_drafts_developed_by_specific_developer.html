Команда пользовательского интерфейса поиска черновиков, разработанных конкретным разработчиком предназначена для поиска черновиков, разработанных конкретным разработчиком. Единственным аргументом команды является знак разработчика. Результатом выполнения команды является вывод всех черновиков указанного разработчика.
