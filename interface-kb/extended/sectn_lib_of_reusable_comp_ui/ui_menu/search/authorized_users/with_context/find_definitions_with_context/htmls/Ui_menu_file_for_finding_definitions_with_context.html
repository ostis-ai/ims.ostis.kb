Команда поиска определения/пояснения для заданного понятия с заданным контекстом предназначена для нахождения определения либо пояснения для заданного понятия в рамках контекста, который ограничивает область поиска агента. Первым аргументом команды является понятие, для которого необходимо найти определение или пояснение. Вторым аргументом является контекст действия, в рамках которого будет осуществляться поиск. Результатом выполнения запроса является множество из определений и пояснений, найденных для понятия с их основными идентификаторами*, само понятие и множество связывающих их дуг с отношением трансляция sc-текста* и атрибутом пример'.
