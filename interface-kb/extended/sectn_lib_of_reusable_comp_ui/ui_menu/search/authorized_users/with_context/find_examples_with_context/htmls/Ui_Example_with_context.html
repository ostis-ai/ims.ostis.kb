Команда поиска примеров для заданного понятия с заданным контекстом предназначена для нахождения примеров заданного понятия. Первым аргументом команды является понятие, для которого необходимо найти примеры. Вторым аргументом является контекст действия, в рамках которого будет осуществляться поиск. Результатом выполнения команды является множество примеров, найденных для понятия, само понятие и множество связывающих их дуг с атрибутом пример'.
