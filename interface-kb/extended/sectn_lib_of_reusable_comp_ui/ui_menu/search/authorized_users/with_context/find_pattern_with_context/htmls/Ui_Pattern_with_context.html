Команда поиска конструкций для заданного шаблона с заданным контекстом предназначена для нахождения всех конструкций, удовлетворяющих заданному шаблону в рамках контекста, который ограничивает область поиска агента. Первым аргументом команды является шаблон, для которого должны быть найдены все соответствующие ему конструкции. Вторым аргументом является контекст действия, в рамках которого будет осуществляться поиск. Результатом выполнения команды является множество найденных для этого шаблона конструкций.
