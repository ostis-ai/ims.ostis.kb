Команда поиска конструкций для заданного шаблона предназначена для нахождения всех конструкций, удовлетворяющих заданному шаблону. Единственным аргументом команды является шаблон, для которого должны быть найдены все соответствующие ему конструкции. Результатом выполнения команды является множество найденных для этого шаблона конструкций.
