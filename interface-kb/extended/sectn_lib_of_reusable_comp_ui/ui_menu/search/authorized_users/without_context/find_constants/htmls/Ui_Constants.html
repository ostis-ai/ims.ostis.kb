Команда поиска понятий, на основе которых определяется заданное понятие предназначена для нахождения понятий, через которые определяется заданное понятие. Единственным аргументом команды является понятие, для которого запрашиваются определяющие его понятия. Результатом выполнения команды является узел, включающий множество понятий, через которые определяется заданное понятие, связанное отношением используемые константы* с заданным понятием.
