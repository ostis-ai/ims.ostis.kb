Команда проверки указания ключевых sc-элементов семантических окрестностей в рамках заданной структуры предназначена для нахождения cемантических окрестностей, у элементов в рамках заданной структуры, у которых не указан ключевой sc-элемент. Единственным аргументом запроса является структура, в которой будет произведена проверка семантические окрестности. Результатом выполнения команды является структура в которой находятся семантические окрестности, у которых не указан ключевой sc-элемент.
