Команда пользовательского интерфейса для проверки наличия максимального класса объектов исследования у предметных областей с заданным контекстом предназначена для проверки наличия максимального класса исследования у предметных областей из заданной структуры в рамках контекста, который ограничивает область видимости агента. Первым аргументом команды является знак структуры, элементы которой необходимо проверить. Вторым аргументом является контекст действия, который будет ограничивать область видимости.
