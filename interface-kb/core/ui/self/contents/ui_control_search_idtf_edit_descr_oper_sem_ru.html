При щелчке левой клавиши мыши по одному из предложенных вариантов, происходит следующее:
  1. при выборе идентификатора, который изображается в виде sc.g-рамки:
    sc.g-рамка
    , то переход осуществляется к семантической окрестности sc-элемента, знаком которого является эта sc.g-рамка. Как правило, это sc.g-рамки, которые содержат идентификаторы, связянные с sc-элементами с помощью отношения . К примеру, такие отношения используются для описания
  2. при выборе идентификатора, рядом с которым присутствует значок , то переход осуществляется к sc-элементу, для которого указанный идентификатор является одним из основных (в одном из режимов идентификации). Другими словами к элеменету, с которым данный идентификатор связан отношением
  3. при выборе идентификатора, рядом с которым присутствует значок , то переход осуществляется к sc-элементу, для которого указанный идентификатор является системным (уникальным в рамках системы). Другими словами к элеменету, с которым данный идентификатор связан отношением
