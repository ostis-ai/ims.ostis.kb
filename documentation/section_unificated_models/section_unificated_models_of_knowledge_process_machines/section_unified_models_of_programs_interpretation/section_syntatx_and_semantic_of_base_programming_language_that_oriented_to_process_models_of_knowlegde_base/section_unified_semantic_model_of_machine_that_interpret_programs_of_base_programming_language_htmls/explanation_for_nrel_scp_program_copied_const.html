Каждая связка отношения копируемые константы scp-программы* связывает sc-узел, обозначающий scp-программу, и множество, содержащее все sc-элементы, которые входят в какой-либо из scp-операторов данной программы как scp-константа', однако являются sc-переменными, и для которых, соответственно, создаются уникальные копии при создании scp-процесса. К таким scp-константам' относятся, помеченные как scp-константа':

  1. множество параметров, передаваемых в вызываемую подпрограмму в scp-операторах асинхронного вызова подпрограммы;
  2. второй и третий операнды scp-операторов генерации sc-конструкции по произвольному образцу и scp-операторов поиска sc-конструкции по произвольному образцу.
