Операторы класса scp-оператор перевода в нижний регистр строкового содержимого файла описывают действие перевода строкового содержимого файла, являющегося значением второго операнда, в нижний регистр.

Каждый оператор данного класса содержит два операнда.

Первый операнд может иметь произвольный тип значения scp-операнда'. Значением данного операнда является файл, идентифицирующий строку, являющееся строкой в верхнем регистре, соответсвующей второму операнду. В случае, если данный операнд помечен как scp-операнд со свободным значением', то файл будет сгенерирован, в противном случае будет изменено содержимое уже имеющегося файла.

Второй операнд должен быть помечен как scp-операнд с заданным значением'. Значением данного операнда является файл, идентифицирующий строку, являющееся аргументом функции перевода строки в нижний регистр.
