Задачей агента подсчёта количества sc-переменных в струтуре является подсчет количества sc-переменных в заданной структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу посчитать количество sc-переменных в структуре. Входным аргументом является структура, для которой необходимо подсчитать количество sc-переменных. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.

    В ходе выполнения агента явно указывается связь со следующими scp-программами:
  • scp-программа генерации ответа для агентов оценки качества;
  • scp-программа добавления сообщения об ошибке;
  • scp-программа подсчёта мощности множества без учёта кратных вхождений;
  • scp-программа подсчёта мощности множества без учёта кратных вхождений в рамках заданного контекста.
