Задачей агента проверки наличия определения у понятия является нахождение понятий, не имеющих ни определения, ни пояснения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия определения или пояснения у понятия. Аргументом запроса является предметная область, у которой необходимо найти понятия, не имеющие ни определения, ни пояснения.
