Задачей агента проверки указания соответствующей предметной области для классов в рамках заданной структуры является нахождение понятий, в заданной структуре, элементы которой не соотнесены с предметной областью. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки указания соответствующей предметной области для классов в рамках заданной структуры. Единственным аргументом запроса является структура, элементы которой необходимо проверить.
