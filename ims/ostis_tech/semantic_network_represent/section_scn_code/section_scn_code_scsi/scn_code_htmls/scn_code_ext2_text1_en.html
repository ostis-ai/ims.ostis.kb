Second SCn-code extension introduces hypermedia capabilities.

Input elements:

  1. Contour sc-identifier, each of which identifies sc-elementdenoting the set of all sc-elements of that sc-graph, which is the result of the translation in SC-code that information which is limited the circuit.
  2. Frame sc-identifier, each of which identifies sc-element (more precisely, sc-link), which is indicated by the file displayed in the box.
  3. Inside contour sc-identifier and frame sc-identifier can be not only linear texts (as in SCs-code), but the information structures of any kind (sc.g-texts, drawings, images). This is the transition to a hypermedia text.
