Каждый черновик представляет собой структуру, описывающую фрагмент базы знаний, разрабатываемый в текущий момент каким-либо разработчиком базы знаний, при этом указанный разработчик является автором* этой структуры. Сама эта структура, ее спецификация и все ее элементы попадают в полную блокировку, соответствующую указанному разработчику базы знаний.
