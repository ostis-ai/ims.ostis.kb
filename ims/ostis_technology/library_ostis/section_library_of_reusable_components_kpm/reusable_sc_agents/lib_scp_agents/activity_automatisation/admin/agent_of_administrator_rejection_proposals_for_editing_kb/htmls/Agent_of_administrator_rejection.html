Задачей агента отклонения предложения по редактированию базы знаний является отклонение* администратором* предложения*, описывающего изменения, вносимые разработчиком* в базу знаний. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу отклонения предложения по редактированию базы знаний. Единственным аргументом запроса является знак действия. рассмотреть предложение по редактированию базы знаний или знак действия. рассмотреть новое проектное задание.
