первичная синтаксическая структура sc-текста* – это бинарное отношение, связывающее некоторый sc-текст с другим sc-текстом, формируемым по следующим правилам:

  • каждому sc-узлу первого sc-текста соответствует синглетон (знак sc-узла) в рамках второго sc-текста;
  • каждому sc-коннектору из первого sc-текста в рамках второго sc-текста соответствует синглетон, обозначающий данный sc-коннектор и соединенный с другими синглетонами второго sc-текста парами инцидентности двух типов, в зависимости от того, началом или концом данного sc-коннектора являются обозначаемые этими синглетонами sc-элементы. В случае, когда sc-коннектор является sc-ребром, то достаточно пар инцидентности первого типа.
  • для каждого синглетона в рамках второго sc-текста явно указывается синтаксический тип, определяемый типом соответствующего ему элемента из первого sc-текста (знак sc-константы, знак sc-узла и т.п.).

Стоит отметить, что подобным образом может быть задана синтаксическая структура любого текста, а не только sc-текста. В этом случае понадобятся другие отношения инцидентности другие классы синтаксических типов.
