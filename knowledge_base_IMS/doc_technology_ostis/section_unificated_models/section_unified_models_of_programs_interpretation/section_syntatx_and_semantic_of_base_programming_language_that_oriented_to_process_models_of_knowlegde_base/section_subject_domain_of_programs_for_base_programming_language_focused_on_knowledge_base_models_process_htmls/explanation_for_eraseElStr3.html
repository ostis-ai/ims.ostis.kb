Операторы класса scp-оператор удаления трехэлементной sc-конструкции описывают действие удаления одного или нескольких sc-элементов, составляющих трехэлементную sc-конструкцию. Каждый оператор данного класса содержит три операнда. При выполнении операторов данного класса выполняется поиск sc-элементов, полностью аналогичный поиску в scp-операторах поиска трехэлементных sc-конструкций, после чего выполняется удаление значений операндов, помеченных как удаляемый sc-элемент'. Если условиям поиска удовлетворяет несколько имеющихся в памяти sc-конструкций, то будут удалены элементы только одной из них.
