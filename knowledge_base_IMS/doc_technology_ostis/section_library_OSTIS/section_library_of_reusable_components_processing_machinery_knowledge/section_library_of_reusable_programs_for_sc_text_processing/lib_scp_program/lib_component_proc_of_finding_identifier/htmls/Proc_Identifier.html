Задачей scp-программы поиска идентификаторов является поиск основных идентификаторов для заданного sc-элемента. Первым входным параметром данной scp-программы является сам sc-элемент, для которого необходимо найти все основные идентификаторы, вторым входным параметром является узел ответа, куда будут занесены все найденные конструкции. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: основным идентификатором* и множеством языков , на которых идентификаторы могут быть написаны.
